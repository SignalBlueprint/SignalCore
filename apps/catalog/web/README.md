# Catalog Web Frontend

React frontend for the Catalog product management system.

## Development

```bash
# Install dependencies (from monorepo root)
pnpm install

# Start dev server (runs on port 5174)
pnpm --filter catalog-web dev

# Build for production
pnpm --filter catalog-web build

# Type check
pnpm --filter catalog-web typecheck
```

## Features

- Product listing and grid view
- AI-powered product upload with image analysis
- Inventory management
- Product search and filtering
- Lookbook management
- Mobile-friendly interface
