# SiteForge

Website generation and management platform that enables teams to quickly create, customize, and deploy demonstration sites and landing pages.

## Purpose

SiteForge provides a streamlined way to build and manage demo sites for products, services, or campaigns. The platform offers templates, customization tools, and deployment automation to reduce the time and effort required to create professional websites.

## Near-Term MVP

- Template-based site generation
- Basic page builder with drag-and-drop functionality
- Site preview and editing capabilities
- Simple deployment to hosting platforms
